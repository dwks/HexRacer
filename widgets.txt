Widgets -- supported observers
    box       -- activated
    text
    button    -- activated
    edit      -- entered
    scrollbar -- scrolled
    listbox   -- selected
    combobox  -- entered, selected
    checkbox  -- toggled

Events
    key (down/up)
    mouse move (focus in/out)
    mouse button (down inside/up inside/up outside)

Observers
    event (any of the above)
    activated (button clicked, checkbox toggled)
    text data entered
    item selected
